Program ini adalah program system pakar untuk
mendeteksi kehamilan, program ini dibuat dengan PHP
dan menggunakan MySQL sebagai databasenya.

Berikut adalah langkah2 instalasi :
1. Letakkan semua file pada htdocs (terserah mau dibuat folder lagi atau ndak).
2. Import file sukasuka.sql dari phpmyadin.
3. Buka file connect.php, kemudian lakukan konfiguras
   pada host, databasename, username, serta password.
4. Jalankan file index.php pada browser.
5. Jika mau login pada halaman admin, gunakan username "admin" dan password "abcd".
